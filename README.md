# Hello World Landing Page

A simple, beautiful landing page project used to test Taskmaster Studio integration.

## Tech Stack
- HTML5
- CSS3 (with modern features)
- Vanilla JavaScript

## Features
- Responsive design
- Dark/Light mode toggle
- Animated hero section
- Contact form
- Smooth scrolling

## Development

This project uses:
- **Taskmaster** for task management
- **Beads** for workflow tracking
- **GitHub** for version control and issues

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. That's it!
